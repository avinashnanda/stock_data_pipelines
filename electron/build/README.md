# Build Resources

This directory holds build resources for electron-builder (icons, etc.).

## Required Icons

For production builds, place your app icons here:

- `icon.ico` – Windows icon (256×256, multi-size ICO)
- `icon.icns` – macOS icon
- `icon.png` – General purpose (512×512 PNG recommended, also used as Linux icon)

If no icons are provided, electron-builder will use a default Electron icon.
